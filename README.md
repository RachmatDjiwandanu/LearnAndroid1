# Project: LearnAndroid1 - Aplikasi Dasar [Nama Spesifik Aplikasi]

## Deskripsi Proyek

Repositori ini adalah bagian dari rangkaian proyek **"LearnAndroid1"**, yang bertujuan untuk mempelajari dasar-dasar pengembangan aplikasi Android.

Direktori `app` ini berisi kode sumber utama untuk proyek pertama (atau saat ini) dalam seri pembelajaran ini. Fokus utama dari proyek ini adalah untuk memahami:

  * Struktur dasar proyek Android Studio.
  * Penggunaan Activity dan Layout XML.
  * [Tambahkan tujuan spesifik lainnya, contoh: Penanganan Input Pengguna Dasar].

## Struktur Direktori `app`

Direktori `app` adalah lokasi standar untuk semua sumber daya dan kode inti aplikasi Android:

| Folder/File | Keterangan |
| :--- | :--- |
| `src/main/java/` | Berisi semua **kode sumber** (Activity, Fragment, Class, dll.) dalam bahasa Kotlin/Java. |
| `src/main/res/` | Berisi semua **sumber daya** aplikasi (layout XML, gambar, string, tema, dll.). |
| `AndroidManifest.xml` | File manifes yang mendefinisikan struktur dan izin aplikasi. |
| `build.gradle` | File konfigurasi build untuk modul aplikasi. |

## Cara Menjalankan Proyek

Untuk menjalankan proyek ini di lingkungan pengembangan lokal Anda, ikuti langkah-langkah berikut:

1.  **Clone Repositori:**
    ```bash
    git clone https://github.com/RachmatDjiwandanu/LearnAndroid1.git
    ```
2.  **Buka di Android Studio:**
      * Buka **Android Studio**.
      * Pilih **Open an existing Android Studio project**.
      * Pilih direktori `LearnAndroid1` yang baru saja Anda clone.
3.  **Sync dan Build:** Tunggu hingga Android Studio menyelesaikan proses *Gradle sync*.
4.  **Jalankan:** Pilih perangkat emulator atau perangkat fisik Anda, lalu tekan tombol **Run** (ikon segitiga hijau).
